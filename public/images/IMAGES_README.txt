DROP YOUR IMAGES HERE
=====================

All images go in this folder: public/images/

------------------------------------------------------------
LOGO
------------------------------------------------------------
File:    logo.png
Format:  PNG with transparent background
Size:    Any width, at least 80px tall (displays at 38px in nav)
Notes:   Should read well on a dark green background

------------------------------------------------------------
FOUNDER PHOTOS (About page carousel)
------------------------------------------------------------
File:    founder-1.jpg   (first founder shown)
File:    founder-2.jpg   (second founder shown)
File:    founder-3.jpg   (third founder shown)
Format:  JPG or PNG
Size:    Minimum 960x840px recommended (displays at 4:3.5 ratio)
Notes:   Faces should be centred. The carousel cycles automatically
         every 4 seconds. Update the founder names and roles in
         about.html by searching for "Founder Name".

------------------------------------------------------------

File:    service-4.jpg   Link Building section
File:    service-5.jpg   SEO Audits section
File:    service-6.jpg   E-commerce SEO section
Format:  JPG
Size:    Minimum 1120x640px recommended (displays at 560x320)
Notes:   If a file is missing the section shows the text label
         instead, so missing images never break the page.

------------------------------------------------------------
PERFORMANCE NOTES
------------------------------------------------------------
- All images are lazy-loaded (loading="lazy") automatically
- Use JPG for photos, PNG for logos/graphics
- Compress images before uploading (use squoosh.app or tinypng.com)
- Target under 200kb per image for fast load times
- WebP format also works if you rename the extension in the HTML

------------------------------------------------------------
UPDATING FOUNDER NAMES
------------------------------------------------------------
Open public/about.html and search for "Founder Name".
There are three instances, one per founder. Replace each with
the real name. The role line below each says
"Co-founder, Honey Bridge SEO" - update that too if needed.
